# Ticket-11 — Microsoft 365 Tenant Setup & Admin Center Navigation

## Objective
Provision a Microsoft 365 tenant and explore the Admin Center interface to understand service administration and user management capabilities.

---

## Lab Environment

- Platform: Microsoft 365 Enterprise Trial
- Tenant Type: Microsoft 365 E5 Trial
- Admin Portal: https://admin.microsoft.com
- License: Microsoft 365 E5 Trial Subscription

---

## Issue Summary

This ticket simulates initial tenant provisioning and administrative familiarization required before performing identity and service management operations within a Microsoft 365 cloud environment.

---

- Registered for Microsoft 365 Enterprise E5 Trial

# IT Support Portfolio

This repository contains simulated Tier 1 IT Support / Service Desk tickets completed in a Windows virtual lab.

## What this project demonstrates
- Windows user account management
- Password reset and login issue resolution
- Troubleshooting methodology
- Documentation like a real IT Service Desk
- Basic security awareness

## Tools Used
- Windows 10 / 11
- VirtualBox
- Computer Management
- Local Users and Groups

## Ticket List
- Ticket 01: User Login / Password Issue
- Ticket 02: Wi-Fi / Network Issue (in progress)
- Ticket 03: Slow PC Performance
- Ticket 04: Software Not Opening
- Ticket 05: Windows Update Error
